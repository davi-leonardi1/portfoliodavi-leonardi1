# 🧠 Classificação Visual — Engenharia de Prompt e Aplicações em IA

Projeto desenvolvido utilizando o **Teachable Machine** com foco em classificação visual baseada em Inteligência Artificial.

O objetivo principal foi realizar análises comparativas entre perfis culturais utilizando imagens de pessoas, explorando diferenças e semelhanças entre características associadas às culturas italiana e francesa.

Durante o desenvolvimento, foram aplicados conceitos de:
- Classificação de imagens
- Treinamento de modelos visuais
- Engenharia de Prompt
- Viés algorítmico
- Análise crítica de IA

O sistema foi treinado com um conjunto limitado de imagens, permitindo observar como a quantidade e diversidade dos dados impactam diretamente na precisão das classificações realizadas pela IA.

---

## ⚠️ Viés Algorítmico

Foi identificado que a utilização de poucas imagens no treinamento pode gerar classificações imprecisas e enviesadas.

Exemplo:
- Pessoas brasileiras com descendência italiana apresentaram maior tendência de serem classificadas como italianas pelo sistema.

---

## 🛠️ Estratégia de Melhoria

Para reduzir erros e aumentar a confiabilidade:
- Ampliação da base de treinamento
- Maior diversidade de imagens
- Refinamento do modelo visual
- Balanceamento de dados
