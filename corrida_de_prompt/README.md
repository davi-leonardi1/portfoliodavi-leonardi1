# 🤖 Corrida de Prompt — Engenharia de Prompt e Aplicações em IA

## 📝 Descrição do Projeto

Este projeto foi desenvolvido na disciplina de **Engenharia de Prompt e Aplicações em IA**, com foco na criação e refinamento de prompts para geração de conteúdo utilizando Inteligência Artificial generativa.

O objetivo principal foi explorar diferentes técnicas de engenharia de prompt para produzir resultados criativos, precisos e alinhados às solicitações propostas, abrangendo tanto geração de imagens quanto geração de textos automatizados.

Durante o desenvolvimento, foram realizados experimentos com prompts descritivos, ajuste de contexto, detalhamento visual e refinamento semântico para melhorar a qualidade das respostas produzidas pela IA.

---

## 🎨 Projeto 1 — Geração de Imagem com IA

### 📌 Objetivo

Criar uma imagem utilizando IA generativa a partir do seguinte prompt:

> “Uma imagem de um astronauta estilo barroco tocando violoncelo em Marte.”

O desafio consistiu em estruturar um prompt rico em detalhes artísticos, combinando elementos de:
- Ficção científica
- Estilo barroco
- Instrumentos clássicos
- Ambientação espacial

## ✉️ Projeto 2 — Geração de Texto com IA

### 📌 Objetivo

Desenvolver um prompt para geração automática de texto formal com o seguinte contexto:

> “Um email formal de desculpa de um pirata para um rei.”

O projeto explorou:
- Estrutura textual formal
- Adequação de linguagem
- Contextualização histórica e criativa
- Ajuste de tom e narrativa

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram adquiridos conhecimentos importantes sobre:

- **Estruturação de Prompts:** Como fornecer contexto detalhado para melhorar os resultados da IA.

- **Refinamento Iterativo:** Ajuste progressivo de prompts para aumentar qualidade e precisão.

- **Criatividade Aplicada:** Combinação de elementos distintos para criação de conteúdos originais.

- **Comunicação com IA:** Entendimento sobre como modelos interpretam comandos textuais.
