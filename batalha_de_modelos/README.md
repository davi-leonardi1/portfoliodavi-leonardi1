# 🎬 Sistema de Recomendação de Filmes

## 📝 Descrição do Projeto
Este projeto consiste em um motor de recomendação inteligente que utiliza técnicas de filtragem colaborativa e baseada em conteúdo. O objetivo principal é diminuir a “paralisia de escolha”, oferecendo sugestões personalizadas com base no histórico de visualização e nas preferências de gênero dos usuários.

Desenvolvido como parte da disciplina de **Inteligência Artificial (2024.1)**, o sistema processa grandes volumes de dados de filmes e avaliações para identificar padrões de comportamento e similaridade entre títulos, utilizando algoritmos de aprendizado de máquina para prever a nota que um usuário daria a filmes ainda não assistidos.

## 📊 Resultados e Aprendizados

O projeto apresentou resultados positivos em ambiente de testes, demonstrando a eficiência do modelo híbrido de recomendação.

- ✅ O sistema atingiu **92% de acurácia** nos testes de validação.
- ✅ Aplicação de técnicas de limpeza e normalização de dados para reduzir ruídos.
- ✅ Implementação de gráficos para visualização de clusters e afinidade entre filmes.
- ✅ Melhor compreensão sobre recomendação baseada em similaridade e comportamento do usuário.
